# Amazon_sales-dashboard
📊 Amazon Sales Dashboard – Power BI Project
📌 Project Overview

This project presents an interactive Amazon Sales Dashboard built using Power BI, designed to analyze sales performance across products, regions, and time. The dashboard provides actionable insights to support data-driven business decisions.

🎯 Objectives
Analyze overall sales performance and revenue trends
Identify top-performing products and regions
Monitor order quantity and pricing patterns
Enable dynamic filtering using slicers for better exploration

📂 Dataset Description

The dataset used in this project contains the following fields:

Order ID – Unique identifier for each order
Order Date – Date of purchase
Product Name – Name of the product sold
Region – Sales region
Price – Price per unit
Quantity – Number of units sold
Total Amount – Total revenue (Price × Quantity)

📈 Key KPIs
💰 Total Revenue
📦 Total Orders
🛒 Total Quantity Sold
📊 Average Order Value (AOV)
📊 Dashboard Features

KPI Cards for quick business insights
Bar Chart – Top products by revenue
Column Chart – Sales by region
Line Chart – Sales trend over time
Pie Chart / Donut Chart – Revenue distribution
Slicers – Filter by date, region, and product
